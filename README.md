# Shiho - Task Manager Chatbot

Shiho is a chatbot application built in Java, developed as part of an individual project for NUS CS2103T (Software Engineering).

## Features:
- Add, list, and delete tasks
- Add, list, and delete notes
- Mark and unmark tasks as completed
- Find tasks by keyword
- Supports different task types: todos, deadlines, and events

**Setup:** JDK 17 required. Open in IntelliJ and run Shiho.main().
